Avnzor feedback & response emails (CRM): merge fields and copy (English / Arabic)

Reference for implementing avnzor-feedback-email-CRM.html (invite / survey request) and avnzor-feedback-response-email-CRM.html (thank you + copy of answers). Replace each {{FIELD_NAME}} in the HTML with your platform’s merge syntax, or with the values below for static sends. Document language and direction: same tokens for both; see section 1.

Survey link: {{CTA_URL}} must be the real feedback/survey URL (often the same in both languages if the form is bilingual). Set this per campaign or per contact, depending on your tool.

1. Document & language

Use these together so the email renders LTR in English and RTL in Arabic.

Field (placeholder) English Arabic
{{HTML_LANG}}enar
{{HTML_DIR}}ltrrtl
{{BODY_DIR}}ltrrtl
{{BODY_LANG}}enar
{{WRAPPER_LANG}}enar
{{WRAPPER_DIR}}ltrrtl
{{CONTENT_LANG}}enar

2. Main copy (invite email only: avnzor-feedback-email-CRM.html)

The survey response template reuses the same field names for the shell ({{H1}}, {{BODY_LEAD}}, features, sign-off, footer) but you supply copy appropriate to that message; see section 5. It has no CTA block.

{{CTA_URL}} Use your live survey URL (https://…). The same link can work for both languages if the form is bilingual.

5. Survey response email (avnzor-feedback-response-email-CRM.html)

Use the same section 1 values for {{HTML_LANG}}, {{HTML_DIR}}, {{BODY_*}}, {{WRAPPER_*}}, and {{CONTENT_LANG}}. Reuse sections 3–4 for features and footer. There is no {{CTA_URL}} or {{CTA_LABEL}} in this file.
