Feedback email templates

Internal reference for campaign and CRM teams. Download ready-to-use HTML, or the merge version with placeholders and the field table for tools such as HubSpot, Salesforce, or your ESP.

About this page

This site hosts three deliverables: a final English message, a final Arabic (RTL) message, and a single CRM template with {{placeholders}} plus a separate field guide (English/Arabic example text). Use the sections below to preview or download each file, and follow the short setup notes inside each block.

Templates

1 English (LTR) template

File

avnzor-feedback-email.html · LTR, English copy only

About this template

Fixed layout: header, body with headline and CTA, feature strip, footer with social, contact, and center logo. Images and links point to live Avnzor assets and channels. No merge tags; use as-is for English sends or as a design reference.

How to use

  1. Import the file into your email or CRM (HTML content block) or hand it to your email developer.
  2. Point the CTA to your real survey or feedback form URL in the code.
  3. Test in major clients (Gmail, Outlook) before scheduling the campaign.

2 Arabic (RTL) template

File

avnzor-feedback-email-ar.html · RTL, Arabic copy

About this template

Same structure as the English file with dir="rtl" and lang="ar" set on the document, wrapper, and Content-Language metadata. Contact lines that must stay LTR (phone, email) use a dedicated class. No merge tags.

How to use

  1. Import the HTML the same way as the English file; your ESP may let you set message language to Arabic for analytics.
  2. Update the CTA to your real survey or bilingual form link.
  3. Run an inbox test for RTL: Gmail, Apple Mail, and Outlook in web and desktop.

3 CRM template and placeholders

Files

avnzor-feedback-email-CRM.html (merge) · avnzor-feedback-email-CRM-FIELDS.html (field guide)

About these files

The CRM file reuses the same design but replaces translatable text, document language, and the survey CTA with {{...}} tags. One file can back both English and Arabic if your tool fills lang and dir plus copy per audience. The field guide lists every token with example English and Arabic phrasing, for mapping into HubSpot, Salesforce, Marketo, or similar. Social, contact, and static icon URLs are hardcoded in the template.

How to use

  1. Open the field guide and copy or import it into a spreadsheet or Google Doc for localization handoff.
  2. Wire each {{placeholder}} in the CRM file to a platform merge field, or do a one-time replace for a static test.
  3. Set {{CTA_URL}} to the live feedback or survey link (per send or per contact, depending on your data).
  4. After merge, test send to internal addresses before the full send.

Field reference (EN/AR): open guide or download the guide as HTML.
